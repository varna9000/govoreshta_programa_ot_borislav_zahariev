# Говореща програма от Борислав Захариев

**Python порт на легендарния български речев синтезатор за Правец 8 (края на 80-те години)**

```
 ГОВОРЕЩА . . . ПРОГРАМА бз-з-ззт ОТ БОРИСЛАВ ЗАХАРИЕВ
```

## Какво е това?

В края на 80-те години Борислав Захариев създава първия български синтезатор на реч — програма за компютъра **Правец 8** (клонинг на Apple II), която може да изговаря текст на български език. С характерния си роботизиран глас, програмата се превръща в емблематична част от българската компютърна история.

За незрящите хора в България тя е много повече от технологична новост — чак до 2003 г. синтезаторът на Захариев остава основното средство за достъп до компютри за хората с увредено зрение у нас.

Този проект е **Python порт**, създаден чрез обратно инженерство (reverse engineering) на оригиналния 6502 машинен код, извлечен от дисково изображение `GOVOR.dsk`.

## Как работи оригиналът?

Правец 8 има само **1-битов говорител** — единственото нещо, което софтуерът може да направи, е да го включва и изключва чрез запис на адрес `$C030`. Захариев кодира всяка фонема като таблица с вълнова форма, в която всеки байт задава времето между превключванията на говорителя. Така от обикновена правоъгълна вълна се получават звуци, наподобяващи формантите на човешката реч.

В оригиналния код има два режима на синтез:

- **Нибъл режим** (за съгласни — с, ш, т, ч, ф, х…): всеки байт се разделя на две 4-битови стойности, управляващи по един полупериод на говорителя.
- **Байтов режим** (за гласни — а, е, и, о, у…): целият байт е една стойност за времетраенето на превключването.

Целият речев двигател заема около **6 KB** машинен код за 6502, съдържащ 30 български фонеми.

Не са необходими допълнителни библиотеки — само Python 3:

```bash
cd govor
python3 govor.py --demo
```

## Употреба

```bash
# Изговори текст (латинска транслитерация) и запиши WAV файл
python3 govor.py "zdrasti"

# Приема и кирилица
python3 govor.py "здрасти"

# Класическото представяне
python3 govor.py --demo

# Промени скоростта (1–12, по подразбиране 8)
python3 govor.py -s 5 "bawno goworene"

# Запиши в конкретен файл
python3 govor.py -o pozdrav.wav "dobar den"

# Интерактивен режим
python3 govor.py -i

# Покажи таблицата с фонемите
python3 govor.py --info

# Генерирай всички фонеми в един файл
python3 govor.py --all
```

## Транслитерация

Оригиналната програма използва специфична латинска кодировка за българските звуци. Python портът приема и кирилица (автоматично се транслитерира), и оригиналната латиница:

| Латиница | Кирилица | Звук | | Латиница | Кирилица | Звук |
|:---------:|:--------:|:----:|---|:---------:|:--------:|:----:|
| `a` | а | а | | `o` | о | о |
| `b` | б | б | | `p` | п | п |
| `c` | ц | ц | | `q` | я | я |
| `d` | д | д | | `r` | р | р |
| `e` | е | е | | `s` | с | с |
| `f` | ф | ф | | `t` | т | т |
| `g` | г | г | | `u` | у | у |
| `h` | х | х | | `v` | в | в |
| `i` | и | и | | `w` | ъ | ъ |
| `j` | ж | ж | | `y` | ь | ь/й |
| `k` | к | к | | `z` | з | з |
| `l` | л | л | | `[` | ш | ш |
| `m` | м | м | | `]` | щ | щ |
| `n` | н | н | | `^` | ч | ч |
| | | | | `` ` `` | ю | ю |

В текста могат да се използват и специални символи:

- `/` — повишава интонацията (намалява скоростта)
- `:` — понижава интонацията (увеличава скоростта)
- `0`–`9` — задава скорост директно (съответства на 3–12)

## Структура на оригиналния двоичен код

Дисковото изображение `GOVOR.dsk` е стандартна 140 KB дискета за DOS 3.3, съдържаща два файла:

```
 A 002 HELLO         ← зарежда програмата: PRINT CHR$(4);"RUN gowor"
 A 026 gowor         ← основната програма
```

Файлът `gowor` е формално Applesoft BASIC програма, но съдържа само един ред:

```basic
0 CALL 2347
```

…след който следва чист 6502 машинен код. `CALL 2347` (`$092B`) прехвърля управлението директно към двигателя за синтез.

### Карта на паметта

| Адрес | Съдържание |
|-------|------------|
| `$0801–$080C` | BASIC стъб (ред 0 CALL 2347) |
| `$080D–$092A` | Разбор на текст, обработка на команди |
| `$092B–$096D` | Инициализация (вход) |
| `$09AD–$0A19` | Ядро на синтеза — превключване на говорителя |
| `$0A1B–$0A8F` | Диспечер на `&` командите |
| `$0AF0–$0B6F` | Имена на числата (нула, едно, две…) |
| `$0B70–$0BFF` | Таблица с указатели към фонемите (по 4 байта) |
| `$0C00–$1DFF` | Вълнови данни за фонемите (~4.6 KB) |
| `$1E01–$1FFF` | Начален екран, помощна информация |

## История

Борислав Захариев създава „Говорещата програма" в края на 80-те години за компютъра Правец 8. Програмата е разпространявана безплатно и се превръща в основно средство за достъпност за незрящите хора в България. По-късно Торос Хованесиян добавя поддръжка на английски език, а Хюсеин Исмаил създава скрийнрийдъра „Ехо", който работи заедно със синтезатора.

В училищните зали и компютърните клубове из цяла България поколения ученици чуват характерния роботизиран глас:

> ГОВОРЕЩА . . . ПРОГРАМА бз-з-ззт ОТ БОРИСЛАВ ЗАХАРИЕВ

Този Python порт е създаден с цел запазване и популяризиране на тази важна част от българската компютърна история.
